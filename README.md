# Obstacle Avoidance using A3C Reinforcement Learning

## Overview
This project implements obstacle avoidance for a mobile robot using the A3C reinforcement learning algorithm.

## Features
- GridWorld simulation
- A3C neural network (Actor-Critic)
- Training and testing modules
- Reward-based learning

## Files
- environment.py → Simulation environment
- a3c_agent.py → Neural network model
- train.py → Training process
- test_model.py → Model testing

## Results
The model successfully learns to reach the goal while avoiding obstacles.

## Tools Used
- Python
- PyTorch
- Matplotlib
